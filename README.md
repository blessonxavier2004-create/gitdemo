Basic Arithmetic Calculator:
A simple, beginner-friendly Python script that performs the four fundamental mathematical operations: addition, subtraction, multiplication, and division.
 Description:
This project demonstrates the basics of Python programming, including:

User Input: Capturing data using the input() function.

Type Casting: Converting string inputs into integers using int().

Arithmetic Operators: Using +, -, *, and /.

Output: Formatting and displaying results to the console.
